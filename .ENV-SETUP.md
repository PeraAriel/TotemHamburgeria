## üö® Problemi Comuni

### "Connection refused"
1. Verificare che Flask sia avviato
2. Verificare che il database sia raggiungibile
3. Verificare URL in .env

### "No such file .env"
```bash
cp .env.example .env
# o
cp .env.dev .env
```

### Database non trovato
1. Verificare credenziali in .env
2. Eseguire `init_db.sql`
3. Controllare che MySQL sia avviato

---

## üìù Note Importanti

1. **NON** commitare file .env in git (sono ignorati)
2. Usare .env.example e .env.dev come template
3. Ogni sviluppatore ha le proprie credenziali .env
4. Per testing, usare .env.dev come base
5. Per produzione, creare .env.prod separato
